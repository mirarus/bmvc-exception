## Installation

Install using composer:

```bash
composer require mirarus/bmvc-exception
```

## Example

Install using composer:

```bash
<?php

	require_once __DIR__ . '/vendor/autoload.php';

	use BMVC\Exception\ClassException;

	throw new ClassException('BMVC');
?>
```
